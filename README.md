# React TypeScript Starter

Initially a stateless starter react app stucture

## Package versions:

### `react: ^16.13.1`

### `typescript: ~3.7.2`

### `react-router-dom: ^5.1.2`

### `node-sass: ^4.13.1`

## Available Scripts

In the project directory, you can run:

### `yarn install`

### `yarn start`

### `yarn test`

### `yarn build`

### `yarn eject`
